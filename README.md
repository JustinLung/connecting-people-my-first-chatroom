## Installatie & Gebruik
```
# Cloning the repository
git clone https://github.com/JustinLung/connecting-people-my-first-chatroom.git

# Moving to the folder
cd connecting-people-my-first-chatroom

# Install NPM packages
npm install

# Run local server
npm run start
```
